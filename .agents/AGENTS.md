# Rules

- **Auto Commit & Push**: Sau khi hoàn thành bất kỳ thao tác chỉnh sửa code nào, tự động chạy lệnh `git add .`, `git commit -m "..."` và `git push` lên GitHub repository `anti-hunchback` mà không cần hỏi lại.
- **Verification**: Claude Code và Codex sẽ kiểm tra những thứ bạn làm, hãy luôn đảm bảo chất lượng code và tuân thủ các quy tắc.
